PROJECT: Job Market Trend Analyzer (with Role-Based Skill Gap Analysis)

We are building a system that analyzes job postings to identify trending skills and perform role-specific skill gap analysis.

------------------------------------

рҹҺҜ CORE FEATURES:

1. Trending Skills Analyzer
- Input: Job dataset
- Output: Top trending skills across market

2. Role-Based Skill Gap Analyzer (IMPORTANT)
- Input: User skills + job role (e.g., "Java Developer")
- Output: Missing skills ONLY for that role

Example:
User: ["java"]
Role: "java developer"

System should NOT suggest:
вқҢ python, вқҢ sql (if not relevant)

Instead suggest:
вң… spring, вң… microservices, etc.

рҹ‘ү This makes the system intelligent and realistic

3. Text Analyzer (Optional)
- Extract skills from job description

------------------------------------

We FILTER jobs by role first.

Flow:

вҡҷпёҸ IMPORTANT FUNCTION (ROLE FILTERING):

In preprocess.py:

def load_data(role=None):
    df = pd.read_csv("data/postings.csv", nrows=5000, encoding="latin1")

    df = df.dropna(subset=["description", "title"])

    if role:
        df = df[df["title"].str.lower().str.contains(role.lower())]

    return df["description"].tolist()
